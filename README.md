# おこづかいクエスト 🪙

子ども向けの、貯金がゲーム感覚で楽しくなるお小遣いアプリです。

- 定額おこづかい（毎月／毎週・支給日に「うけとる」ボタン）
- お手伝いクエストで報酬をゲット
- 欲しいもの（目標）の達成率を見える化
- レベルアップ・バッジ・紙吹雪演出
- 複数人（きょうだい）対応
- データは各端末のブラウザ内に保存（サーバー不要）

## 使い方
スマホやPCのブラウザで `index.html` を開くだけ。GitHub Pages に公開すれば、URL でどこからでも使えます。

## 技術
バニラ HTML / CSS / JavaScript の単一ファイル。localStorage 保存・オフライン動作。
